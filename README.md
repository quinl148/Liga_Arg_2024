# Liga Argentina Football Season 2024 Dataset

This is a dataset created from scraped data from https://fbref.com/en/comps/21/2024/schedule/2024-Liga-Profesional-Argentina-Scores-and-Fixtures. 

# Dimensions

* game_id
* week
* day
* date
* time
* venue
* match report
* team
* opponent
* home_away
* formation
* match_outcome
* record
